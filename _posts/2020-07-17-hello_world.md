```plain
  最最开始的时候，小饭团并不叫小饭团。当然，这得追溯回他刚刚在Daddy肚子里报道的时候了，那时候的他还是它，还叫小三角。
  看着手中的白纸黑字，两个突然之间被安排了新鲜角色的年轻爸爸相视无言，愣了好一小会儿。要一起创造一个小三角什么的，虽然会在夜深情动之时被当作活跃气氛的调侃话语说出来，但两人都未曾想到，这个意外之喜就这么来到了他们的生活之中。
  虽然其中确实也有措施没做到位的因素。
  开车回家的路上，斑鸠三角时不时瞟向坐在副驾驶座上紧紧捏着报告单的三好一成。看着对方紧皱的眉头，三角犹豫了一会儿，忍不住小心翼翼地问道：“阿成，留下它好不好？”
  正处于事业上升期的Omega难免有些惊慌不安，认真思索了一阵后，最终还是微笑着点了点头“嗯，留下它吧！回去后也得和夏组的大家还有监督说一下这件事呢Piko~”
  不出所料地收获了众人祝福，不过，与此同时，小三角这个名字也被狠狠吐槽了一番。
  “虽然是好事，但是三角元素超标了吧！那么多三角真的分得清吗？”
  经过一番叽叽喳喳的讨论后，“小饭团”这个昵称成功脱引而出，被当事AO拍板定下了。毕竟，作为小三角的进阶版，小饭团这个词既能够显示三角对三角的热衷，又能够反映一成对美食的热衷。伴随着“和咖喱也是绝配！”的画外音，Omega体内的小细胞被正式赋予了这么个名字。
  数月之后，某个阳光明媚的夏日正午，小饭团正式降临在了这个丰富多彩的美好世界。
```
2.
```plain
  自婴儿时期起，小饭团就极为让人省心。性格温和、乖巧，基本上不会哭鼻子，更不用说闹觉和起床气了。在大部分时间里，他都是该吃吃、该喝喝、没事干就笑呵呵的状态。即使偶尔被蹲在摇篮旁的两个爸爸闹醒了美梦，他也只是皱着鼻子哼哼两声，等过了一小会儿，一睁开眼，就又回到了喜笑颜开的模式。
  也有那么几次，三好一成把还没睡够的小饭团抱在怀里哄。这下小孩儿到不会哼哼了，但是依旧皱着鼻子，嗅着Daddy信息素的味道往Omega的胸口贴去，闭着眼拱一拱、蹭一蹭，还会撒娇地扭一扭头。守在一旁的斑鸠三角通常会眼疾手快地塞一个温热的奶瓶过去，小饭团也不恼，只是将它乖乖接住，抱在怀里心满意足地开始喝起来。等奶瓶见了底，就睁开绿色的双眸冲Papa和Daddy咯咯咯地笑。
```
3.
```plain
  闲暇时间里，两个新上任的爸爸也会用自己的方式带着小孩子认识这个世界。
  小饭团刚刚学会爬没多久，三好一成就把白嫩嫩肉乎乎的小婴儿抱到了自己的画室里。往地上铺开一张白色画布，握着小小的手掌轻轻压在颜料盘上，再按下一个个色彩缤纷的手印。最开始的时候，小饭团还有些不适应颜料在手掌上滑腻腻的感觉，但在被Daddy带着进行了几下“艺术创作”后，他便开始喜欢上了这种玩耍方式。慢慢的，他开始大胆了起来，试着自已一个人去体验，用不同颜色印出自己的指纹、掌印，然后回头看向微笑着给他拍照的Daddy。
  天气好的时候，斑鸠三角也会把小饭团领到附近的公园和猫咪们打招呼，教他怎么和猫咪聊天，哪怕怀中的小婴儿连话都说不清楚，只会笑着咿咿呀呀几声，顺带再挥几下手表示自己的喜悦之情。或许是与猫咪们相交甚好的关系，尽管小饭团在看到大部分毛茸茸好的动物时都会明显兴奋起来，却对猫猫们和各种猫咪玩偶有着独特的情感，以至于当他开始说话时，第一个蹦出来的词就是“喵呜～”
```
4.
```plain
  当然，除了爸爸们之外，小饭团也享受了了很多来自其他人的关爱。
  幸和椋会定期在放学后结伴跑到家里来，一个会把自己亲自做的可爱小衣服套在小孩身上，另一个则会给小孩读各种绘本和童话故事。九门和天马来的时候，安静祥和的场面则会变得热闹起来。小饭团喜欢被九门抛得高高地然后接住，也喜欢坐在天马肩上骑马马。尽管大明星十次有八次都会露出一脸尴尬不知所措的表情，但最终还是会心甘情愿的蹲下，让肉团子爬到他的背上。
  夏组全员也曾经感慨过：小饭团不哭不闹也不吵人，日常就是睁着水汪汪的大眼睛安静地观察着大家的一举一动一言一行，和他的两个爸爸简直截然相反。对此，三好一成一次又一次地尝试去解释，说自己高中前也是这么个安静的性格，所以小饭团绝对是随了他。
  当然，这方面随没随不知道，但小饭团在捧场方面可谓得到了两人真传。
  一次一起聚餐的时候，看着桌上的蔬菜咖喱，立花泉两眼发光，赞不绝口，滔滔不绝地开启了新一轮咖喱讲座。
  “咖喱就是……”
  “赛高！”
  总结性发言突然被一个稚嫩的声音打断，习惯了唱独角戏的立花往旁边一看，发现被喂着米糊糊的小饭团看了看她盘中的咖喱，又看了看她，最后还鼓起了掌。忽略了周围人明显憋笑的表情，兴奋不已的立花当场决定要收一个闭门弟子，好将咖喱文化发扬光大。
```
5.
```plain
  或许每个小朋友都会问出那么几个相似的问题，小饭团也不例外，从幼儿园回来后，他蹬蹬跑到两个爸爸面前，仰着头问道：“我究竟是怎么来的呀？”
  “是Papa向三角之神许愿送来的Piko～”
  “是Daddy画出来的哦～”
  意料之外的，得到了两个全然不同答案。脑子有些转不过来的小饭团一脸迷惑。他究竟是被送来的，还是被画出来的呢？
  一成摸了摸小饭团浅蓝色的头发，问道：“你觉得你是怎么来的呢？”
  小孩儿左思右想，犹豫着回答道：“我希望两个都是真的诶……”
  “为什么呢？”
  “因为三角之神很厉害，三角形最棒了！但是，Daddy画的东西也都好好看哦，Daddy最好了！我也想要有那么好看，好纠结……”
  “这样啊，”三角一把将两人搂在怀中，“你原本是Daddy画出来的哦，然后Papa向三角之神许了个愿，你就从画里跳出来啦！”
  一切都解释得通了！小饭团恍然大悟，原来是这样子啊。
```

```plain
  斑鸠三角注意到今天的小饭团好像有什么烦恼。从幼儿园回来之后就独自翻起了家里的相册，一边看一边叹气。
   “怎么了吗？”端着一盘饭团走到小孩旁边，三角轻声问道。
  小饭团指了指大合照上的身影，带着沮丧的哭腔说道：“Papa有弟弟，Daddy有妹妹，可是我什么都没有……”
  “有Papa和Daddy陪你不好吗？”
  “当然好！”小孩子理直气壮地回答道，随后又叹了口气，“可是我也好想当哥哥啊……”
  看着小饭团低落的样子，三角有些哭笑不得，毕竟这也不是他能做主的事情。
  “那你想要弟弟还是妹妹啊？”
  “我都想要，”祖母绿的眼睛转了一圈，似乎是想到了什么新的解决方案，小饭团整个人的兴致突然起来了。“Papa，你说如果Daddy把弟弟妹妹画出来，我再向三角之神许愿，是不是我就有弟弟妹妹了！”
  三角假装思索了一下，然后点了点头，说：“或许可以哦！但三角之神只会实现乖孩子的愿望，所以你要乖乖吃饭不能挑食，要按时睡觉，也要听我和Daddy的话哦。”
  看着小团子疯狂点头的模样，斑鸠三角忽然意识到自己忘了告诉小饭团，整件事中最重要的一环，其实在于如何小孩能如何劝动他Daddy，让一成愿意把弟弟妹妹画出来吧。
```
7.
```plain
  尴尬的事情也不是没有发生过。
  一番雨沾云惹后，三角覆在一成身上调整着呼吸，他吻了吻Omega的腺体，又轻轻咬了一口，将自己的信息素输了进去。气氛被再次点燃，一成搭在三角肩上的双手又开始上下作动，卧室的门却被敲响了，这使得身居下位的一成一下子僵硬起来，一时间不知如何是好。
  三角先是用被单撒了擦自己脸上的汗，又翻身伸手将被扔在地板上的内裤捡起来，套在一成身上。这才起了身去开门，看见小饭团正抱着一个大大的超级三角君站在门口。
  “Papa！我今晚可以和你们一起睡吗？”
  啊，这……看着那和恋人如出一辙的亮晶晶大眼睛，尽管知道有些不妥，三角却没办法说出拒绝的话语。事实上，还未等他开口，小家伙就已经绕过他往床边窜去了。
  不好！
  “Daddy肚子鼓鼓诶！是偷偷吃了夜宵吗？”
  三好一成还没从余韵中彻底缓过神来，就听到身边粘人的小家伙传来的关切问候。一时间羞愧得有些手足无措，只想钻进被子里就这么遁了，还好屋子里一片漆黑，小饭团看不到自己发红的脸。
  “不可以压哦！”察觉到小饭团企图将手伸向一成略微鼓起的小腹后，三角一步跨上床将小孩一把抱起，放在了床的另一边，继而接着说道：“Daddy只是有点不舒服啦，我们陪着他一起睡好不好？”
  “那我要挨着Daddy睡～我还要亲亲Daddy，让Daddy快点好起来！”小孩奶声奶气地撒娇道。
  趁着三角把小饭团重新抱到身边的空档，一成也从被窝里钻了出来，在幼童肉乎乎的脸颊上轻轻落下一个吻，引得小孩发出一阵轻笑。
  “但是记得不能碰Daddy的肚子哦，不然Daddy就不愿意给你画弟弟和妹妹了。”
  “角角！”
  “哇，我要弟弟和妹妹～”
  看着躺在自己身边热火朝天地讨论着相关事宜的一大一小，三好一成不禁回想起这段时间两人一直在自己耳边喋喋叨叨的样子，聊着关于要弟弟妹妹这件事。此时此刻，三好一成只觉得自己就要被磨得没想法了。
  那么，究竟该怎么办呢？
```
